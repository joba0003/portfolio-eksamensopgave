Tema 02 - Web

I dette tema lærte vi om de mest anvendte redskaber for en multimediedesigner. Vi lærte at sætte hjemmesider op vha. HTML og CSS. Vi lærte desuden også om Visual Studio Code samt designkonventioner og gestaltprincipper. Det var vores første rigtige hjemmeside, lavet ud fra givet materiale.

DESIGNKONVENTIONER

Vi lærte om HTML og CSS og fik de første færdigheder inde for grafik og opsætning af tekst og billeder i Figma. Vi lærte også om filformater til web (at ændre dem fra jpg til webp). Vi arbejdede desuden også med designkonventioner og gestaltprincipper. Såsom nærhed, lighed, lukkethed og forbundethed. Der lærte jeg meget om brugervenlighed igennem de forskellige love, og har brugt dem meget siden.

GIVET MATERIALE

Det vi især lærte på dette tema, var at udarbejde en opgave ud fra et oplæg. Vi arbejdede med givet materiale samt noget kort indhold vi selv skabte. Vi talte også meget om ophavsret og rettighedslicenser, vi fik vist nogle sider, hvor der er gratis billeder uden ophavsret. Vi talte også lidt om metadata, som er den data der ligger bag billedet, det kunne fx være hvor billedet blev taget og hvem der ejer det.

PROGRAMMER

Vi blev introduceret for Visual Studio Code, som er et kodnings-program til HTML og CSS. Vi lærte også at overføre filer til filezilla og HTML-validering, så der ikke er fejl i vores kode, det samme med CSS-validering. Så lærte vi også om inspektoren i Firefox og Chrome. Vi blev også introduceret for squoosh.com, som er en af mine yndlings sider, da man kan ændre filformat og beskære og det hele på én gang.
